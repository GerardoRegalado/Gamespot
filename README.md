
# Frontend Developer Test Trial.

The objective of this project is to develop a responsive web application for the GameSpot platform using React, TypeScript, Redux, and various supporting libraries to handle the user interface and API requests.

## Run Locally

Clone the project

```bash
  git clone https://github.com/GerardoRegalado/Gamespot
```

Go to the project directory

```bash
  cd gamespot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
